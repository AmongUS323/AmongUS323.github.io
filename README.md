# Portfolio Site (Drop-in for GitHub Pages)

This folder is a **static website** (no build step). You can copy it directly into a GitHub repo and publish with GitHub Pages.

## Quick start

1. Put these files in the root of your repo:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `assets/` (images)

2. Replace the placeholder images:
   - `assets/hero.jpg`
   - `assets/headshot.jpg`

3. Edit content in `index.html`:
   - email
   - intro paragraphs
   - “Select Works” cards (title, tags, link)

## Publish on GitHub Pages

- In your repo: **Settings → Pages**
- Source: **Deploy from a branch**
- Branch: `main` (or `master`) / folder: `/root`

Your site will appear at:
- `https://<username>.github.io/<repo>/`

## Tip: custom domain or root site
If you want the site at `https://<username>.github.io/`, name the repo `<username>.github.io`.
